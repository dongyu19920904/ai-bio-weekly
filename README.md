## 日刊特色

- 📌 **今天的三句话**  
  用 2～3 句话讲清楚：今天在忙什么、卡在哪、情绪如何。

- 🎬 **今天的瞬间（2～3 个）**  
  从日记里挑几个有画面的片段：学习、卖账号、工具试用、散步聊天等。

- 🧰 **工具 / 概念小卡片（1～2 张）**  
  只写我当天真的打开过、查过、折腾过的 AI 或生命科学相关工具 / 概念，用白话解释。

- 📮 **留给明天的一点小话**  
  给未来的自己留一两句提醒，再附 1～3 个可以立刻行动的小任务。

阅读体验：**单篇约 1–3 分钟**，适合当成刷手机时的一张小卡片。
